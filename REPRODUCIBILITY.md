# Reproducibility guide

## 1. Data preparation

Set `DATASET_ROOT` to the folder that contains the `derivatives/` directory:

```bash
export DATASET_ROOT=/path/to/dataset/root
```

The real EEG loader uses:

- `*_eeg-epo.fif` for MNE epochs.
- `*_events.dat` for class and condition labels.
- `*_report.pkl` for optional contamination metadata.

## 2. Synthetic experiments

```bash
bash scripts/run_synthetic_main.sh
```

The experiment manifest is located at:

```text
synthetic_benchmarks/configs/experiments/experiment_manifest.yaml
```

## 3. Proposed method on real EEG

```bash
DATASET_ROOT=$DATASET_ROOT DEVICE=cuda bash scripts/run_real_eeg_fixed_sub01.sh
```

The fixed-reference protocol is defined in:

```text
real_eeg_remida/configs/fixed_sub01_all_subjects.yaml
```

## 4. Baseline experiments

```bash
DATASET_ROOT=$DATASET_ROOT bash scripts/build_baseline_npz.sh
bash scripts/run_baselines_fixed_sub01.sh
```

This runs nine fixed-reference target tasks:

```text
sub-01 -> sub-02
sub-01 -> sub-03
...
sub-01 -> sub-10
```

The baseline scripts save `run_meta.json`, `split_indices.npz`, per-input metrics, and aggregate CSV/JSON summaries. TCL reuses the corresponding iVAE split so that both baselines use the same `reftrain`, `refval`, `targetadapt`, and `targettest` partitions.

## 5. Result records

The repository includes the fixed-reference baseline summary used in the manuscript:

```text
results/baseline_fixed_sub01/overall_mean_std.csv
results/baseline_fixed_sub01/per_pair_metrics.csv
results/baseline_fixed_sub01/test_read.summary.json
```

Large intermediate artifacts are regenerated by the scripts and are excluded from version control.
