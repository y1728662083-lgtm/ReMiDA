# Result records

This directory stores submitted result records that accompany the code. Large intermediate feature files, checkpoints, exported `.fif` files, and raw EEG data are regenerated by the scripts and are not included in the repository.

## Included records

- `baseline_fixed_sub01/overall_mean_std.csv`: aggregate TCL/iVAE baseline metrics for the fixed `sub-01` reference protocol.
- `baseline_fixed_sub01/per_pair_metrics.csv`: per-target-subject TCL/iVAE baseline metrics.
- `baseline_fixed_sub01/test_read.summary.json`: metadata summary of the baseline `.npz` data extraction.
- `synthetic_reference/table_lin_base_ds0p2.txt`: representative synthetic linear-drift log record.
